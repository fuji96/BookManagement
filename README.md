BookManagement
=====

ユーザー登録したメンバーで、書籍の登録・レビューの投稿・共有ができます。

## Books

書籍の登録ができます。
以下の項目を記入して保存してください。
```
　・書籍名
　・著者
　・値段
　・出版日
```

また、その書籍について投稿されているレビューを閲覧することができます。


## Reviews

レビューの投稿ができます。
登録されている書籍の一覧から一冊を選び、それについての感想を記述してください。


## Users

登録しているユーザーの一覧を確認できます。

また、そのユーザーがレビューを投稿した書籍の一覧と、投稿したレビューの一覧が確認できます。。
